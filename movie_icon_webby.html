3D DVD Screen

Design Brief

The brief for this engineering project was as follows: "develop a reliable working prototype of an interactive product that will deliver different forms of feedback and motion to elicit an emotion."

Project Description

Our project took the approach of creating a mechanical implementation of the DVD loading screen as seen below. A user will press a button to start the game of chance, similar to a roulette wheel, i.e. there is a random chance of winning based on a random allocaiton of time that the machine runs, where a win is denoted by the DVD logo hitting the corner.
